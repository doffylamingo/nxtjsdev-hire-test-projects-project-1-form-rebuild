Donation Form -- Starter

I will create a main `MultiStepForm` component that keeps track of the current step using `useState`. I will store all form field values in one shared state object in this parent component so the data doesn’t get lost when moving between steps. Each step of the form will be separated into its own component to keep the code cleaner and easier to read. The parent component will handle the Next and Back buttons and only show the Submit button on the final step. I will also add a simple progress indicator at the top to show which step the user is currently on.
